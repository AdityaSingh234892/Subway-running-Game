# Quick Start Guide - Subway Runner 3D

## Running the Game in VS Code with Live Server

### Option 1: Using Live Server Extension (Recommended)
1. Open the project folder in VS Code
2. Install "Live Server" extension if not already installed
3. Right-click on `index.html` in the Explorer
4. Select "Open with Live Server"
5. Game opens automatically in your browser

### Option 2: Direct Browser Opening
1. Double-click `index.html` in File Explorer
2. Or drag `index.html` into any modern browser window

## Controls
- **← →** Arrow Keys: Move between lanes
- **↑** Arrow Up: Jump over obstacles  
- **↓** Arrow Down: Slide under obstacles
- **Space**: Alternative jump key
- **ESC**: Pause/Resume game

## Game Features
✅ Player runs automatically forward  
✅ 3 lanes with smooth switching  
✅ Obstacles: trains, barriers, signals  
✅ Coin collection system  
✅ Score increases with distance and coins  
✅ Increasing speed over time  
✅ Collision detection (game over on hit)  
✅ Basic animations (jump, slide, run)  
✅ Railway track environment  
✅ Start screen & Game over screen  
✅ Score display UI  

## Troubleshooting
If game doesn't load:
1. Check browser console (F12) for errors
2. Ensure internet connection (loads Three.js from CDN)
3. Try Chrome/Firefox for best WebGL support
4. Clear browser cache if issues persist
